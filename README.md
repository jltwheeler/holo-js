# HOLO js

A 3d audio visualiser project inspired by Eric Prydz HOLO live set.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
